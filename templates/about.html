Hi, I'm Eric and welcome to my dev site where I share some of the personal projects I'm working on. Recently, I've been investigating what all this hype is about surrounding htmx and I've got to say — I'm a fan! By now, I've worked in all the popular javascript frameworks such as React, Svelte, etc., but there's something very satisfying about using a minimal setup and returning the power back to the server. I built this site using Axum, a Rust server on the backend, Askama, a Rust template engine, htmx and tailwindcss to add reactivity and style for the frontend, and hosted it all on fly.io in a Docker container.

In my free time, I enjoy playing all racket sports such as tennis, pickleball, badminton, and table tennis. I also enjoy paddleboarding, working out, and maintaining a healthy lifestyle.
